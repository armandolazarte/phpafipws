Librería para la gestión de WebServices de la Agencia de Recaudación y Control Aduanero (ARCA ex AFIP).

PhpAfipWs es una pequeña librería que permite que cualquier sistema en PHP pueda conectarse a los servicios de ARCA.

> **Requires [PHP 8.0+](https://php.net/releases/)**

**PhpAfipWs** fue creado por **[Armando Lazarte](https://x.com/ArmandoLazarte)** bajo la **[licencia MIT](https://opensource.org/licenses/MIT)**.
